### Tech
#### Backend
 - Ollama with llama3 model
 - .NET 9
 - Mongodb
 - Redis
#### Frontend
 - Vite React
 - TailwindCSS
 - Docker

### Installation
 #### Server
 - Instale Ollama
 - Use llama3
 - Comece os containers:
   ```
   docker compose up -d
    ```
 #### Frontend
 - Acesse a url no navegador
   ```
    http://127.0.0.1:517
   ```
